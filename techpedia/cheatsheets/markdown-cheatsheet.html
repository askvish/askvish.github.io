Emphasis

You can add emphasis by making text bold or italic.

Bold

Markdown HTML Rendered Output
**bold** text <strong>bold</strong> text bold text
__bold__ text <strong>bold</strong> text bold text
Love**is**bold Love<strong>is</strong>bold Loveisbold

Italic

Markdown HTML Rendered Output
*italic* text <em>italic</em> text italic text
_italic_ text <em>italic</em> text italic text
A*cat*meow A<em>cat</em>meow Acatmeow

Bold and Italic

Markdown HTML Rendered Output
***bold and italic*** <em><b>bold and italic</b></em> bold and italic
___bold and italic___ <em><b>bold and italic</b></em> bold and italic
A***cat***meow A<em><b>cat</b></em>meow Acatmeow
